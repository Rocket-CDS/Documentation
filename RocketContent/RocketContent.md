# RocketContent
RocketContent is an open-source extension for the DNN (DotNetNuke) platform, designed to make website content creation more flexible and developer-friendly.

At its core, RocketContent uses AppTheme design, Razor-based templates that control both the administrative interface and the public-facing content. Several example AppThemes are included, built with W3.CSS, enabling you to quickly launch full-featured websites or customize your own design and layout, with your preferred CSS framework.

**Key Features**

ChatGPT Integration – Harness the power of AI to assist in content creation directly within your workflow.

DeepL Language Translation – Seamlessly translate content into multiple languages using DeepL.

Image Management – Upload images with automatic resizing and conversion to modern WebP format for optimal performance.

Document Uploading – Securely manage and display downloadable documents.

Flexible Templating System – Use Razor-based AppThemes and Cascading Style Templates with support for 3 levels of design hierarchy.

Secure File Delivery – Built-in security for accessing and downloading files based on user permissions.

Multiple Row Display - Multiple rows can be added to any AppTheme.
